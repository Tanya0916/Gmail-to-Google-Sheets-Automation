# Gmail to Google Sheets Automation
This project uses Python and Google APIs to fetch emails from Gmail and store them in Google Sheets automatically. 
It helps streamline reporting, logging, and data analysis without manual copy-paste.

Features
->Fetch emails from Gmail using Gmail API.
->Parse subject, sender, date, and body.
->Write structured data into Google Sheets.
->Secure authentication with OAuth2 credentials.
->Reproducible workflow with clear documentation.
